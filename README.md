# FakePlayer

Fake Player: This plugin is a tool dedicated to testing server/plugin functions without the need to use two phones/PCs to have two or more active players on the server.

## Features

- It imitates a real player.
- FakePlayer can run commands 
- click pressing shift to open log

## Comming Features

- Automatic walking
- PvP automatic

## Usage

- /fakeplayer cmd [name] [cmd] (replace spaces by "." example command.arg1.arg2)
- /fakeplayer create [name]
- /fakeplayer remove [name]
- /fakeplayer list
